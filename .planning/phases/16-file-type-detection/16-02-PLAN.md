<task type="auto">
  <name>Task 1: Pass URL to API for server-side detection in install command</name>
  <files>packages/cli/src/commands/install.ts</files>
  <action>
Currently the install command fetches URL content locally and sends only `{ content }` to the API. This means the API never sees the URL and cannot perform URL-based detection.

Change the install command to send both `content` and `url` when the source is a URL:

In the `try` block (around line 118-119), where `auditViaApi({ content })` is called, update to:

```typescript
// Send URL alongside content so API can perform URL-based agent detection
const auditPayload: { content: string; url?: string } =
  resolved.type === "url"
    ? { content, url: resolved.url }
    : { content };
const response = await auditViaApi(auditPayload);
```

Update `auditViaApi` type signature in `api-client.ts` to accept the union with both fields:

```typescript
export async function auditViaApi(
  body: { content: string; url?: string } | { url: string },
): Promise<AuditResponse> {
```

This allows the API route to extract the URL pathname for agent detection while still having the content pre-fetched by the CLI. The API route already handles `content` taking precedence over `url` for fetching, so sending both is safe -- the API will use the content directly and also extract agent info from the URL.

Also apply URL-based detection locally for the case where the API doesn't return a detected_agent (e.g., content-only pasted audits done locally). Import `getAgentForPath` from `@skillgate/shared`. After receiving the audit response, if `response.result.detected_agent` is absent or "unknown" and the source was a URL, detect locally:

```typescript
if (resolved.type === "url" && (!response.result.detected_agent || response.result.detected_agent === "unknown")) {
  try {
    const urlPath = new URL(resolved.url).pathname;
    const localAgent = getAgentForPath(urlPath);
    if (localAgent !== "unknown") {
      response.result.detected_agent = localAgent;
    }
  } catch { /* skip */ }
}
```
