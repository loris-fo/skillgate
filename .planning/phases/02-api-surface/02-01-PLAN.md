Create all shared lib modules in `apps/web/src/lib/`:

1. **types.ts** - API response envelope types:
   ```typescript
   import type { AuditResult } from "@skillgate/audit-engine";

   export type AuditMeta = {
     slug: string;
     url: string;        // /api/report/{slug}
     badge_url: string;  // /api/badge/{slug}.svg
     created_at: string; // ISO 8601
     cached: boolean;
   };

   export type AuditResponse = {
     result: AuditResult;
     meta: AuditMeta;
   };

   export type ApiErrorCode = "RATE_LIMITED" | "NOT_FOUND" | "INPUT_TOO_LARGE" | "API_ERROR" | "VALIDATION_ERROR";

   export type ErrorResponse = {
     error: {
       code: ApiErrorCode;
       message: string;
     };
   };
   ```

2. **errors.ts** - Error response helper:
   - `errorResponse(code: ApiErrorCode, message: string, status: number, headers?: Record<string, string>): Response`
   - Returns `Response.json({ error: { code, message } }, { status, headers })`
   - Maps AuditError codes to HTTP status codes in a helper: INPUT_TOO_LARGE->413, API_ERROR->502, VALIDATION_ERROR->422, RATE_LIMITED->429, NOT_FOUND->404

3. **kv.ts** - Shared Redis singleton:
   - `export const redis = Redis.fromEnv();`
   - Import from `@upstash/redis`
   - Single instance reused across serverless invocations

4. **rate-limit.ts** - Rate limiter:
   - Import redis from ./kv

   - Prefix: `"ratelimit:audit"`
   - Include `ephemeralCache: new Map()` for in-memory optimization

   - `generateBadge(verdict: Verdict): string` - calls makeBadge with label="Skillgate", flat style, labelColor="#555"
