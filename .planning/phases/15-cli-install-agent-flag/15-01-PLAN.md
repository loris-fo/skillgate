In packages/shared/src/agent-map.ts, add a `getInstallPath` function that resolves an agent name to its install directory:

```typescript
/**
 * Returns the install directory for an agent, or null if the agent
 * only uses dotfiles (not directory-based install).
 * Installable = has at least one path with trailing slash in AGENT_SCAN_MAP.
 */
export function getInstallPath(agent: string): string | null {
  const entry = AGENT_SCAN_MAP.find((e) => e.agent === agent);
  if (!entry) return null;
  const dirPath = entry.paths.find((p) => p.endsWith("/"));
  return dirPath ?? null; // null means no directory-based install
}
```

Logic: find the agent's entry in AGENT_SCAN_MAP, return the first path that ends with "/" (Phase 14 convention: trailing slash = directory). Returns null if agent not found OR agent has no directory paths (dotfile-only).

In packages/shared/src/index.ts, add `getInstallPath` to the exports.

Do NOT create a new AGENT_INSTALL_MAP per user decision -- reuse AGENT_SCAN_MAP.
