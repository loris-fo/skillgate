    - scan --path /custom/dir: discovers .md files in specified directory
    - scan with 3 passing skills: prints table with 3 rows, exits 0
    - scan with 1 High skill among 3: prints table, exits 1
    - scan --no-fail with 1 High skill: prints table, exits 0
    - scan --json: outputs JSON array of { file, result } objects to stdout, no table
    - scan with no files found: prints "No skill files found" message, exits 0
    - scan limits concurrency to 5 parallel API calls
    - scan shows rate limit warning when file count exceeds 25
    - scan spinner shows progress: "Auditing skills... (2/5)"
  </behavior>
  <action>
    1. Create src/commands/scan.ts exporting scanCommand(options: { path?: string; fail: boolean; json: boolean }):
       - Discover files: use glob to find **/*.md in target directories. Default dirs: [".claude/", ".claude/skills/"]. If --path provided, use that instead. Resolve all paths relative to process.cwd().
       - If no files found, print message and exit 0.
       - If files.length > 25, print warning: "Warning: scanning {n} files may approach the API rate limit of 30/hour"
       - Create output handler via createOutputHandler(options.json)
       - Start spinner "Auditing skills... (0/{total})"
       - Audit files concurrently with concurrency limit of 5. For each file: read content via fs.readFileSync, call auditViaApi({ content }), update spinner text with progress count. Collect results as Array<{ file: string; response?: AuditResponse; error?: string }>.
       - Simple concurrency limiter: process files in batches of 5 using a pool pattern (Promise-based semaphore, not external library).
       - Stop spinner on completion.
       - Output results: if --json, write JSON array to stdout. Otherwise, call printScanTable(results).
       - Gating: check if any result has isBlocked(result.overall_score). If blocked and options.fail is true (default), exit 1. If --no-fail (options.fail is false), exit 0.
       - NOTE: Commander's --no-fail inverts to options.fail = false. So check options.fail directly.
