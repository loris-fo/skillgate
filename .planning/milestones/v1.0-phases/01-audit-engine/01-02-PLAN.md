     a. **Validate input**: Check content is non-empty after trimming. Check `Buffer.byteLength(content, "utf8") <= 100_000` (100KB limit). Throw AuditError with appropriate code if invalid.
     b. **Build cache key**: `buildCacheKey(content)`.
     c. **Check cache**: `getCached(cacheKey)`. If hit, return immediately.
     d. **Call Claude API**: Create message with model "claude-sonnet-4-20250514", max_tokens 4096, system SYSTEM_PROMPT, tools [AUDIT_TOOL], tool_choice { type: "tool", name: "record_audit" }, messages [{ role: "user", content: buildUserMessage(content) }].
