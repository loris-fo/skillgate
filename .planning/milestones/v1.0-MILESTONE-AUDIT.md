## E2E User Flows (2/2)

### Flow 1: Web Audit
1. User pastes SKILL.md content or URL on homepage → `audit-form.tsx`
2. Form POSTs to `/api/audit` → `route.ts` calls `auditSkill()`
3. API returns `AuditResponse` with `meta.slug`
4. Form redirects to `/report/[slug]` → `page.tsx` loads from Redis
5. Report renders: hero (verdict), 5 category cards (expandable), utility section, badge snippet

**Status:** Complete end-to-end

### Flow 2: CLI Install
1. User runs `skillgate install <source>` → `install.ts`
2. `resolveInput()` detects local/URL/slug → fetches content
3. `auditViaApi({ content })` POSTs to web API
4. `isBlocked()` checks score → exits 1 on High/Critical (unless `--force`)
5. `writeFileSync()` places SKILL.md in `.claude/` directory

**Status:** Complete end-to-end
